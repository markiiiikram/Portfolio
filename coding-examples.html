Example #1

This is an example of my code from Netmatters assessment. I chose this code due to the fact that it uses nesting, CSS selectors and Pseudo Elements.


Firstly, the button uses nesting, which creates a scope for the styles applied to the button. Any changes made to the styles will only affect elements within that specific container (.button-h).


Secondly, The code uses a specific CSS selector. It targets elements with a class attribute that starts with "icon-". The ^ symbol in the selector denotes the starting point, so it selects elements whose class attribute begins with "icon-".


Lastly, the ::before pseudo-element is applied to the selected elements (those with a class starting with "icon-"). Pseudo-elements allow you to style certain parts of an element without adding extra HTML. In this case, ::before is used to apply styles before the content of each selected element.
